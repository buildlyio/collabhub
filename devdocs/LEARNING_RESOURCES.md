# Developer Learning Resources

This directory contains learning resources and documentation for topics covered in the Buildly CollabHub Developer Assessment.

## Core Programming & Development

### [Writing Production-Quality Code](./production-code.md)
Learn best practices for writing clean, maintainable, production-ready code including error handling, logging, testing, and documentation.

- [How to Write Production-Ready Code](https://www.freecodecamp.org/news/how-to-write-production-ready-code/)
- [Clean Code Principles](https://github.com/ryanmcdermott/clean-code-javascript)

### [Debugging Techniques and Tools](./debugging.md)
Master systematic debugging approaches: reproduce issues, isolate problems, inspect state, implement fixes, and verify solutions.

- [Chrome DevTools](https://developers.google.com/web/tools/chrome-devtools/javascript)
- [Debugging Best Practices](https://www.freecodecamp.org/news/debugging-best-practices/)

## How to Use These Resources

1. **Complete the Assessment** - Take the Developer Level Assessment to understand your current skill level
2. **Review Your Results** - Identify areas where you scored lower
3. **Start Learning** - Begin with the fundamentals in areas you're less comfortable with
4. **Practice** - Apply what you learn in real projects on Buildly CollabHub
5. **Get Feedback** - Participate in code reviews and collaborate with more experienced developers
